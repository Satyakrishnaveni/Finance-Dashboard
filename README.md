# 📊 Finance Dashboard

A modern and interactive finance dashboard built using **React.js**.

---

## 🚀 Features

* 📈 Dashboard Summary Cards
* 📊 Balance Trend Chart (Recharts)
* 🍩 Spending Breakdown (Donut Chart)
* 📋 Transactions Table
* 🔍 Filtering & Search
* 👥 Role-based UI (Admin / Viewer)
* 💡 Insights Section
* 🌙 Dark / Light Mode
* 📱 Fully Responsive Design

---

## 🛠 Tech Stack

* React.js
* JavaScript
* CSS
* Recharts

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```
